# MagiConf

This application is in portuguese so the readme is not written in english

O Magiconf é uma aplicação móvel que tem como objectivo melhorar a experiência de um participante em conferências científicas.
É desenvolvida para Android 4.0+ e tem também uma componente servidor (desenvolvida com Django)

As suas principais funcionalidades são:

- Consulta da agenda (sempre actualizada) da conferência
- Construção de agenda personalizada
- Troca de contactos com outros participantes
- Consulta de detalhes da conferência
- Recepção de notificações emitidas pela organização da conferência
- Facilidade de efectuar check-in na conferência
- Facilidade na gestão dos dados da conferência por parte da organização (através do backoffice no servidor)
